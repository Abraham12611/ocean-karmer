A Meteora DLMM Liquidity Sentinel for the Colosseum Agent Hackathon.

## Features

- **Automated Rebalancing**: Monitor and rebalance DLMM positions.
- **Liquidity Providing**: Provide liquidity to specific pools.
- **Risk Management**: Set stop-loss and take-profit levels.

## Tech Stack

- **Framework**: Node.js / TypeScript
- **SDK**: `@meteora-ag/dlmm`
- **Blockchain**: Solana

## Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure environment variables (`.env`).
4. Run the sentinel: `npm start`
